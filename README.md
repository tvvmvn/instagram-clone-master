## HTTP Request & Response
![HTTP](/images/http.png)

### HTTP Request Method
1. GET - Read Data
2. POST - Create Data
3. PUT - Update Data
4. DELETE - Delete Data

### HTTP Response Status Code
1. 1XX (Informational Response)
2. 2XX (Successful)
- 200 OK
3. 3XX (Redirection)
- 304 NotModified
4. 4XX (Client Error)
- 400 BadRequest
- 401 Unauthorized
- 404 NotFound
5. 5XX (Server Error)
- 500 Internal Server Error
